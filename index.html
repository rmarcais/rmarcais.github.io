Features

In this section, you can find some features available in this version.

Create an account

The perfect way to start using Motiv' is to create an acount.The only information you need are a username, a password, your location, the sports you like and a biography (optional). That's all ! You're now a Motiv' user !

Meet friends !

Search for friends in the "Meet friends !" section! You can filter users by username, location (city and department) and sport. By clicking on the "show infos !" button, you can see all the selected user's information such as his reviews, his location (with a map), his events, his sports and his bio.

Search an join events !

If you want to do sport with other people, this section is for you ! Indeed, you can see all the available events and filter them by (like users) title, location and sports !. If you want to see more details about an event, you can click the button "show infos !" and once you want to join it, just click "Join !". Good session !
