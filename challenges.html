Time to code !

Consignes

Maintenant que vous avez appris les bases du HTML et du CSS, il est temps de créer votre propre site web ! Voici vos consignes :

  • Vous devez créer une page d'accueil avec un titre et une description de votre projet.
  • Utilisez des images et des liens pour rendre la page interactive.
  • Ajoutez des styles CSS pour personnaliser votre page (couleur, taille de texte, marges, etc.).

Préparer votre environnement de travail (VSCode)

Voici un tutoriel rapide pour commencer à travailler dans VSCode :

  • Créez un nouveau dossier pour votre projet et ouvrez-le dans VSCode.
  • Créez deux fichiers : index.html et styles.css
  • Dans le fichier index.html, commencez par insérer la structure de base HTML.
  • Dans le fichier styles.css, ajoutez des styles de base pour personnaliser votre page.

Si vous bloquez à certaines étapes, n'hésitez pas à demander de l'aide aux autres groupes, ou bien à vos mentors !

Pour bien démarrer...

Voici le début de votre fichier index.html. Vous devez maintenant compléter le contenu à l'intérieur de la balise <body> :

    <!DOCTYPE html>
    <html lang="fr">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>Votre projet - Page d'accueil</title>
        <link rel="stylesheet" href="styles.css">
    </head>
    <body>
        
    </body>
    </html>

Testez votre site

Une fois que vous avez ajouté les éléments ci-dessus dans vos fichiers index.html et styles.css, ouvrez le fichier index.html dans un navigateur pour voir votre page d'accueil en action !

Vérification

Assurez-vous que votre site est conforme aux consignes suivantes :

  • La page doit contenir un titre et une description de votre projet.
  • La navigation doit être présente avec des liens fonctionnels.
  • Le design doit être propre et structuré, avec des marges appropriées et des couleurs harmonieuses.

Présentation devant les jurés

À partir de 15h, chaque groupe devra présenter son site web devant un jury. Voici quelques consignes pour bien préparer votre présentation :

  • Assurez-vous que votre site est complet et fonctionnel.
  • Préparez une brève présentation expliquant les choix que vous avez faits pour la création de votre site (structure, design, fonctionnalités, etc.).
  • Chaque membre du groupe devra expliquer une partie du projet lors de la présentation.
  • Vous serez jugés sur la créativité, la qualité du code, et l'originalité du site.

Bonne chance, et n'oubliez pas de vous entraîner pour être à l'aise lors de votre présentation !
