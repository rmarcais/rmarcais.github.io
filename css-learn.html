Qu'est-ce que le CSS ?

Le CSS, ou Cascading Style Sheets (feuilles de style en cascade), est le langage utilisé pour styliser et mettre en forme une page web. Si le HTML est la structure de la page, le CSS est ce qui permet de la rendre beau, attrayant et lisible pour l'utilisateur.

Une analogie : Le CSS, c’est comme la décoration intérieure d'une maison

Imaginez que vous avez une maison (qui représente votre site web). Le HTML est la structure de base : les murs, les fenêtres, les portes. Mais pour que la maison soit confortable et jolie, vous devez ajouter des éléments de décoration : peinture sur les murs, meubles, rideaux, etc. Ces éléments décoratifs sont comme le CSS, qui permet de personnaliser l'apparence de votre site.

Avec le CSS, vous pouvez changer la couleur des éléments, la taille de la police, l'espacement entre les éléments, ajouter des bordures, des ombres, et bien plus encore !

Les règles de CSS : Les instructions pour l'apparence

Le CSS fonctionne avec des règles. Une règle CSS est comme une instruction qui dit à votre page web comment un élément doit apparaître. Par exemple, vous pouvez dire "tous les titres doivent être rouges" ou "les paragraphes doivent avoir une police de caractère plus grande".

Une règle CSS est composée de deux parties :

  • Le sélecteur : Cela indique l'élément auquel la règle s'applique (par exemple, tous les titres <h1>).
  • La déclaration : C'est ce qui définit l'apparence de l'élément (par exemple, la couleur ou la taille du texte).

Par exemple, une règle CSS pourrait ressembler à ceci :

    h1 {
        color: blue;
        font-size: 32px;
    }

Cette règle dit que tous les titres <h1> doivent être affichés en bleu et avec une taille de police de 32 pixels.

Pourquoi le CSS est-il important ?

Sans CSS, votre site web serait un peu comme une maison sans décoration : fonctionnelle, mais peu attrayante. Le CSS est ce qui donne vie à votre site et le rend agréable à regarder. Il permet d'améliorer l'expérience de l'utilisateur en rendant la navigation plus claire et plus esthétique.

En résumé, le CSS transforme un site de base en un site élégant et attrayant, tout comme la décoration transforme une maison fonctionnelle en un endroit où l'on se sent bien.

Couleur du texte : color

La propriété color permet de changer la couleur du texte.

    h1 {
        color: #3498db; /* Applique une couleur bleue */
    }

Consigne : Changez la couleur du texte du titre principal <h1> en vert.

Couleur de fond : background-color

La propriété background-color permet de définir la couleur de fond d'un élément.

    body {
        background-color: #f0f0f0; /* Applique un fond gris clair */
    }

Consigne : Changez la couleur de fond de la page (balise <body>) en bleu clair.

Marge : margin

La propriété margin permet d'ajouter de l'espace autour des éléments.

    p {
        margin: 20px; /* Applique une marge de 20px autour du paragraphe */
    }

Consigne : Ajoutez une marge de 30px autour du paragraphe de la section introduction.

Espacement intérieur : padding

La propriété padding ajoute de l'espace à l'intérieur des éléments, entre le bord et le contenu.

    ul {
        padding: 10px; /* Ajoute un espacement intérieur de 10px autour de la liste */
    }

Consigne : Ajoutez un espacement intérieur de 15px autour de la liste des étapes.

Police de caractères : font-family

La propriété font-family permet de définir la police de caractères utilisée pour le texte.

    h1 {
        font-family: 'Arial', sans-serif; /* Applique la police Arial au titre */
    }

Consigne : Changez la police de la citation (balise <blockquote>) en 'Courier New'.

Taille de la police : font-size

La propriété font-size permet de définir la taille du texte.

    h1 {
        font-size: 36px; /* Change la taille du titre en 36px */
    }

Consigne : Changez la taille du texte du titre <h1> à 48px.

Alignement du texte : text-align

La propriété text-align permet de contrôler l'alignement du texte.

    p {
        text-align: center; /* Aligne le texte au centre */
    }

Consigne : Alignez le texte du paragraphe introductif à droite.

Bordure : border

La propriété border permet de créer une bordure autour des éléments.

    blockquote {
        border: 2px solid #3498db; /* Ajoute une bordure bleue autour de la citation */
    }

Consigne : Ajoutez une bordure noire de 3px autour de la liste des étapes.

Arrondir les coins : border-radius

La propriété border-radius permet de donner des coins arrondis aux éléments.

    ul {
        border-radius: 10px; /* Arrondir les coins de la liste */
    }

Consigne : Arrondissez les coins de la citation à 20px.

Masquer les puces de la liste : list-style-type

La propriété list-style-type permet de modifier ou de supprimer les puces des listes.

    ul {
        list-style-type: none; /* Supprime les puces de la liste */
    }

Consigne : Supprimez les puces de la liste des étapes et mettez une couleur de fond différente pour chaque élément.

Hauteur de ligne : line-height

La propriété line-height permet d'ajuster l'espace entre les lignes de texte.

    p {
        line-height: 1.8; /* Augmente l'espacement entre les lignes du paragraphe */
    }

Consigne : Augmentez l'espace entre les lignes du paragraphe introductif à 2.0.

Transformation : transform

La propriété transform permet de transformer un élément en 2D ou 3D (rotation, échelle, etc.).

    h1 {
        transform: rotate(10deg); /* Tourne le titre de 10 degrés */
    }

Consigne : Faites pivoter le titre principal de 15 degrés.

Ombre portée : box-shadow

La propriété box-shadow permet d'ajouter une ombre portée autour des éléments.

    h1 {
        box-shadow: 5px 5px 10px rgba(0, 0, 0, 0.3); /* Ombre légère autour du titre */
    }

Consigne : Ajoutez une ombre légère à la citation.

Opacité : opacity

La propriété opacity permet de rendre un élément plus ou moins transparent.

    h1 {
        opacity: 0.8; /* Applique une opacité de 80% au titre */
    }

Consigne : Rendre la citation légèrement transparente en utilisant opacity.

Grille CSS : display: grid

La propriété display: grid permet de créer une grille pour disposer les éléments.

    .container {
        display: grid;
        grid-template-columns: repeat(3, 1fr); /* Crée 3 colonnes égales */
    }

Consigne : Créez une grille pour afficher trois boîtes avec du texte, chacune ayant une largeur égale.

Mettez en pratique ce que vous avez appris !

Vous avez ci dessous une page HTML de base. Voici les consignes pour personnaliser cette page avec du CSS :

  • Le titre principal : Mets-le en couleur bleue et change sa taille pour qu'il soit grand.
  • Le paragraphe introductif : Change la couleur du texte en bleu clair et mets-le en italique.
  • La liste des étapes : Affiche-la sans les puces, en utilisant une couleur de fond différente pour chaque élément.
  • Les citations : Change leur police pour quelque chose d'extra, et ajoute une bordure autour d'elles.
