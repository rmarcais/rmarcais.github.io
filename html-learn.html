Qu'est-ce que le HTML ?

HTML, ou HyperText Markup Language, est comme la fondation ou l'ossature d'un site web. Imaginez que vous construisez une maison. HTML serait la structure de base : les murs, le toit, les fondations et la disposition générale de la maison.

Une analogie : Le HTML, c’est comme un plan de construction

Pour construire une maison, vous avez besoin de plans qui indiquent où placer les différentes pièces : chambres, salon, cuisine, etc. De la même manière, le HTML crée un plan pour une page web en indiquant où chaque élément doit être placé.

En HTML, chaque partie d'une page web (un titre, un texte, une image) est comme une pièce de la maison. Le HTML vous aide à organiser tout cela pour que tout soit à sa place et que le site soit lisible pour l'utilisateur.

Les balises HTML : Les instructions pour le navigateur

Les balises HTML sont comme des étiquettes que l'on met sur chaque pièce de la maison. Ces étiquettes expliquent au navigateur comment chaque élément doit être présenté. Par exemple, une étiquette pourrait dire : "Cette partie est un titre", ou "Cette partie est un paragraphe de texte".

Le HTML n'est pas responsable de l'apparence de votre site web (c'est le rôle du CSS). Il sert uniquement à organiser les éléments et à leur donner un sens, comme le plan de construction qui définit où chaque pièce de la maison doit se trouver.

Pourquoi le HTML est-il important ?

Sans HTML, il n'y aurait pas de structure de base pour votre site web. Tout le contenu serait mélangé, comme si vous essayiez de vivre dans une maison sans murs ni portes. HTML organise votre contenu de manière à ce que le navigateur puisse l’afficher correctement et que l'utilisateur puisse naviguer facilement sur votre site.

Découvrons à présent les différentes balises que vous allez pouvoir utiliser...

<html>

La balise <html> définit le début et la fin de la page web. C’est la balise principale qui englobe toute la structure de la page.

<html> ... </html>

<head>

La balise <head> contient des informations importantes sur la page, mais qui ne sont pas visibles sur l'écran, comme le titre, les liens vers les feuilles de style, et les métadonnées.

<head> ... </head>

<body>

La balise <body> englobe tout le contenu visible de la page, comme les textes, images, vidéos, et autres éléments interactifs.

<body> ... </body>

<title>

La balise <title> définit le titre de la page, qui est affiché dans l'onglet du navigateur.

<title>Mon super site</title>

<h1> à <h6>

Les balises <h1> à <h6> définissent les titres, du plus important (<h1>) au moins important (<h6>).

    <h1>Mon titre principal</h1>
    <h2>Sous-titre</h2>

<p>

La balise <p> est utilisée pour créer des paragraphes de texte.

<p>Voici un exemple de paragraphe.</p>

<a>

La balise <a> permet de créer un lien cliquable vers une autre page ou un autre site web.

<a href="https://www.example.com">Cliquez ici</a>

<img>

La balise <img> permet d'ajouter une image à la page. L'attribut src définit le chemin de l'image, et alt fournit une description de l'image pour l'accessibilité.

<img src="image.jpg" alt="Description de l'image">

<ul> et <ol>

Les balises <ul> (liste non ordonnée) et <ol> (liste ordonnée) permettent de créer des listes d'éléments. Les éléments de liste sont définis avec <li>.

    <ul>
        <li>Élément 1</li>
        <li>Élément 2</li>
    </ul>
    
    <ol>
        <li>Premier</li>
        <li>Deuxième</li>
    </ol>

<div>

La balise <div> est un élément de regroupement. Elle est utilisée pour structurer des parties du contenu de la page et appliquer des styles.

<div>Contenu ici</div>

<span>

La balise <span> est utilisée pour appliquer des styles ou des modifications de manière ciblée sur une portion de texte dans une ligne, sans créer de saut de ligne.

<span>Texte modifié</span>

<form>

La balise <form> permet de créer un formulaire. Elle contient des éléments comme les champs de texte, les boutons et les cases à cocher pour permettre à l'utilisateur d'envoyer des données.

    <form>
        <input type="text" placeholder="Votre nom">
        <button type="submit">Envoyer</button>
    </form>

<input>

La balise <input> permet de créer des champs interactifs dans un formulaire, comme des zones de texte, des boutons radio, des cases à cocher, etc.

<input type="text" placeholder="Nom">

<button>

La balise <button> permet de créer un bouton cliquable.

<button>Cliquez-moi</button>

<strong>

La balise <strong> est utilisée pour rendre un texte important (habituellement en gras).

<strong>Texte important</strong>

<em>

La balise <em> est utilisée pour mettre en évidence un texte par italique.

<em>Texte en italique</em>

<br>

La balise <br> permet de créer un saut de ligne, utile pour formater le texte dans un paragraphe sans créer un nouveau bloc de contenu.

<p>Texte avant le saut<br>Texte après le saut</p>
